RESTful Routes

A table of all 5 RESTful routes

Name Path HTTP Verb Purpose Mongoose Method
Index /api/todos GET List all todos Todo.find()
Create /api/todos POST Create a new todo, then redirect somewhere Todo.create()
Show /api/todos/:todoId GET Show info about one specific todo Todo.findById()
Update /api/todos/:todoId PUT Update particular todo, then redirect somewhere Todo.findByIdAndUpdate()
Destroy /api/todos/:todoId DELETE Delete a particular todo, then redirect somewhere Todo.findByIdAndRemove()
